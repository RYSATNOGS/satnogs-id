# satnogs-id
Post-launch object identification. 
After a rideshare drops 50 cubesats at once, figuring out which object is which — and refining TLEs from Doppler across stations — is genuinely hard and time-sensitive. 
SatNOGS observation data is uniquely positioned to help. 
This is the most research-flavored and highest-prestige, but also the riskiest to actually finish.
